
# App Icon Instructions

## Creating the Notebook Icon

To complete the app setup, you need to create a notebook icon image file:

**File Location:** `assets/images/notebook-icon.png`

**Specifications:**
- Size: 1024x1024 pixels (for best quality)
- Format: PNG with transparency
- Design: A romantic notebook icon with:
  - A closed notebook/journal appearance
  - Pink/rose color scheme (#E91E63 or similar)
  - Optional: A heart symbol or romantic decoration
  - Clean, simple design that works at small sizes

**Quick Options:**

1. **Use an AI Image Generator:**
   - Prompt: "A simple, flat design icon of a pink romantic notebook with a heart, 1024x1024, transparent background, app icon style"

2. **Use Figma/Canva:**
   - Create a 1024x1024 canvas
   - Design a simple notebook shape
   - Add romantic elements (hearts, ribbons)
   - Export as PNG

3. **Use Icon Generator Websites:**
   - Visit: https://www.appicon.co/ or similar
   - Upload or create your notebook design
   - Download the generated icon

4. **Temporary Solution:**
   - You can temporarily use the existing `natively-dark.png` icon
   - Copy it and rename to `notebook-icon.png`

Once you have the icon, place it at: `assets/images/notebook-icon.png`

The app is configured to use this icon for:
- App launcher icon (Android & iOS)
- Splash screen
- Notifications
- Web favicon
